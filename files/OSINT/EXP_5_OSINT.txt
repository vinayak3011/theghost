Q1. What is Reverse Image Analysis?
A. Reverse Image Analysis is a technique used to trace the origin, location, or source of an image using tools that analyze metadata or search for visually similar images online.

Q2. What does OSINT stand for?
A. OSINT stands for Open-Source Intelligence, which involves collecting and analyzing publicly available information for investigative or research purposes.

Q3. Which tools are commonly used for Reverse Image Analysis?
A. Google Lens, TinEye, Yandex Image Search, Duplichecker, and Bing Visual Search are common tools used.

Q4. What is EXIF metadata?
A. EXIF (Exchangeable Image File Format) metadata is data automatically stored in an image file by a camera or smartphone that includes details like date, time, camera type, and GPS coordinates.

Q5. What type of information can you extract from EXIF metadata?
A. You can extract the device model, camera settings, image resolution, GPS location, and capture date/time.

Q6. How can GPS coordinates from metadata help in OSINT investigations?
A. GPS coordinates help pinpoint the exact geographical location where the photo was taken, which is crucial in forensic or intelligence investigations.

Q7. Name two websites that can display metadata from an image.
A. https://ezgif.com/view-metadata
 and https://exif.regex.info/exif.cgi

Q8. What is the difference between metadata analysis and reverse image search?
A. Metadata analysis extracts internal image data (like GPS info), while reverse image search compares the visual appearance of the image to find similar or identical images on the internet.

Q9. What is the purpose of using Google Lens in this experiment?
A. Google Lens is used to visually analyze the image and find matching or similar images to determine its location or origin.

Q10. What is the significance of the website ‚Äúgeoimgr.com‚Äù in this experiment?
A. Geoimgr.com provides sample photos with embedded GPS metadata, which can be used to practice extracting geolocation information.

Q11. What kind of real-world applications does reverse image analysis have?
A. It is used in digital forensics, verifying fake news, social media investigations, locating missing persons, and copyright verification.

Q12. What happens if GPS metadata is not available in an image?
A. If GPS data is missing, investigators rely on visual analysis like landmarks, signs, and architecture using reverse image search tools.

Q13. Why do some people remove metadata from their images?
A. For privacy and security reasons, to prevent others from finding their location or device information.

Q14. What are the limitations of reverse image analysis?
A. Limitations include missing metadata, edited or cropped images, poor image quality, and unavailability of matching images online.

Q15. How can reverse image analysis contribute to cyber investigations?
A. It helps identify fake accounts, detect misinformation, trace image origins, and locate suspects or victims in digital crime cases.

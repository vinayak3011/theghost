DevSecOps Exp 7

Aim: create Application and code security testing using snyk.



Steps to perform
open kali
# Open your Kali Linux VM/installation (start the machine or open the OS).

open terminal
# Open a terminal window to run all following commands.

	sudo apt update && sudo apt upgrade
# Update APT package lists and upgrade installed packages to latest available versions.

	sudo apt install -y curl git ca-certificates build-essential
# Install essential utilities: curl (download), git (VCS), ca-certificates (TLS trust), and build tools.

	sudo apt install -y nodejs npm
# Install Node.js runtime and npm package manager (needed for Snyk CLI and Node projects).

	node -v
# Print the installed Node.js version to verify installation.

	npm -v
# Print the installed npm version to verify installation.

	sudo apt install -y python3 python3-venv python3-pip
# Install Python 3, virtual environment support, and pip (Python package manager).

	python3 -m venv ~/snyk-venv
# Create an isolated Python virtual environment at ~/snyk-venv for Python tooling.

	source ~/snyk-venv/bin/activate
# Activate the Python virtual environment so pip installs go inside it.

	pip install --upgrade pip setuptools
# Upgrade pip and setuptools inside the venv to latest versions for reliability.

	sudo apt install -y docker.io
# Install Docker engine (required if you plan to scan container images).

	sudo systemctl enable --now docker
# Enable Docker to start on boot and start the Docker service now.

	sudo usermod -aG docker $USER
# Add your user to the 'docker' group so you can run docker without sudo (log out/in required).

	sudo npm install -g snyk
# Install Snyk CLI globally using npm (quick and common install method).

	snyk --version
# Verify Snyk CLI installed correctly by printing its version.

	curl -L https://downloads.snyk.io/cli/latest/snyk-linux -o snyk
# Download the Snyk standalone Linux binary (alternative to npm install).

	chmod +x snyk
# Make the downloaded Snyk binary executable.

	sudo mv snyk /usr/local/bin/snyk
# Move the Snyk binary into a system PATH location so you can run 'snyk' from anywhere.

	snyk --version
# Verify the Snyk binary is installed and print the version.

	snyk auth
# Authenticate the Snyk CLI to your Snyk account (opens a browser for OAuth login).

	mkdir snyk-lab-node && cd snyk-lab-node
# Create a test Node project directory and change into it.

	npm init -y
# Generate a default package.json for the Node project (auto answers).

	npm install express lodash
# Install example dependencies (express web framework and lodash utility library).

	npm install debug@2.6.9
# Intentionally install an older version of 'debug' (example vulnerable dep) for demo scans.

	snyk test
# Run a local Snyk vulnerability scan for the project and show found issues in the terminal.

	snyk monitor
# Send a snapshot of the project to Snyk's cloud to track the project over time and view results in the web UI.
